# 🚦 Smart City RAG Assistant

This project is an AI-powered Smart City Intelligence Assistant that analyzes traffic and pollution data using a Retrieval-Augmented Generation (RAG) approach.

## Features
- Upload traffic dataset
- Visualize traffic and pollution trends
- Analyze congestion and pollution correlation
- Ask natural language questions
- Get intelligent insights and suggestions

## Tech Stack
- Python
- Streamlit
- Pandas
- Plotly
- Scikit-learn (TF-IDF based retrieval)

## Example Questions
- Which area has highest traffic?
- Which location has highest pollution?
- How does traffic affect pollution?
- Give suggestions to reduce congestion
